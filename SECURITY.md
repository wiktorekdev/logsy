# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please email wiktorekdev@gmail.com.

We will acknowledge receipt within 48 hours and provide updates on the fix progress.
